Creating a Plugin Project

For new projects, it is highly recommended to use Gradle-based setup.

This section explains how you can create a new plugin project from scratch using the wizard. Optionally, you can import an existing project or import a project from external models. You can also add a new plugin module to the current IntelliJ Platform project. For more information, refer to the IntelliJ IDEA Web Help.
